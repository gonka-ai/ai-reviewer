# Decentralized API
This is a server that runs along side the blockchain node. It's purposes are:
1. Provide a REST API for delivering AI inferences
2. Register AI inferences on the blockchain
3. Monitor the blockchain and validate other particiapnts AI inferences when needed
4. Orchestrate the ML nodes that will be used by the participant
5. Orchestrate periodic "Proof of Compute" (PoC) work to prove how much compute a participant has brought to the network.
